## Este repositorio

Mi portfolio personal, **hecho a mano con HTML5, CSS3 y JavaScript vanilla** — cero frameworks, cero dependencias, listo para cualquier hosting.

| | |
|---|---|
| 🌗 **Dos personas, un interruptor** | El modo claro muestra a la desarrolladora (retrato inspeccionado en DevTools); el modo oscuro, a la artista (retrato B&N a pantalla completa). |
| 🌍 **Bilingüe ES / EN** | Traducción instantánea de toda la web sin recargar, con la preferencia guardada. |
| 📄 **CV imprimible en A4** | `cv.html` genera un CV de una sola hoja, solo en versión clara con colores que ahorran tinta ♻ |
| ⚡ **Detalles de programadora** | Terminal `whoami` autoescrito, timeline en formato `git log`, barra de estado de editor, animaciones con respeto a `prefers-reduced-motion`. |
